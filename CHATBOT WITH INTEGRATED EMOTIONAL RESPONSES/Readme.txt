Emotion-Aware Chatbot
This repository contains the source code for an Emotion-Aware Chatbot that adjusts its responses based on the emotions detected in the user's input. The chatbot uses machine learning models to analyze user sentiment and generate appropriate replies tailored to the emotional context (e.g., happy, sad, angry, neutral).

Features
Emotion Detection: The chatbot detects emotions such as happiness, sadness, anger, and neutral tones using NLP-based sentiment analysis.
Contextual Responses: The chatbot tailors its replies to fit the user's emotional state, providing supportive responses for negative emotions and positive reinforcement for happy emotions.
Machine Learning Model: Pre-trained model for emotion classification based on text input.
Real-time Interaction: The chatbot engages with users in real-time, making it suitable for applications like mental health support, customer service, and conversational agents.
